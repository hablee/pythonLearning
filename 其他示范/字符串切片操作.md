# 利用负数获取温度的数值，因为是两位数还是个位数不确定，所以从前面不好切
```python
wd1 = '34C'
sz1 = wd1[0:-1]
wd2 = '4C'
sz2 = wd2[0:-1]
print(sz1)
print(sz2)
```
# 如果要切片取到最后一个元素，最后用`:`
