一只猴子第一天摘下若干个桃子，当即吃了一半，还不过瘾，又多吃了一个；第二天早上又将剩下的桃子吃掉一半，又多吃了一个。
以后每天早上都吃了前一天剩下的一半加一个。到第N天早上想再吃时，见只剩下一个桃子了。问：第一天共摘了多少个桃子？
# 输入格式:
输入在一行中给出正整数N（1<N≤10）。
# 输出格式:
在一行中输出第一天共摘了多少个桃子。
# 输入样例:
`3`
# 输出样例:
`10`
# 解答
```python
N = int(input())
m=2*(N-1)
k = m
while(m!=1):
        m=k+2
        k = m
        for i in range(N-1):
                m=m-m/2-1
                if m<=0:
                        break
print(k)
```
# 测试结果
答案正确
# 思路
如果不满足最后一天等于1的条件，就一直往上加。值得注意的是，桃子的总数肯定是个偶数
